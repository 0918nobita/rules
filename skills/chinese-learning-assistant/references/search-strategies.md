# Web検索での用例探しガイド

中国語の実用例を効果的に見つけるための検索戦略とテクニック。

---

## Web検索を使う目的

### 1. 実際の使用例を見つける
- ネイティブスピーカーの自然な表現
- 文脈付きの用例
- 最新の使用例

### 2. 使用頻度を確認する
- よく使われる表現か
- 現代でも使われているか
- どの場面でよく使われるか

### 3. 自然さを検証する
- 提案した表現が実際に使われているか
- ネイティブの感覚を確認

### 4. バリエーションを発見する
- 似た表現
- 地域差
- 文脈による違い

---

## 検索キーワードの作り方

### 基本パターン

#### パターン1: 完全一致検索

```
"[表現]"
```

例:
- "有点儿意思"
- "挺好的"
- "搞定了"

**用途**: 正確にその表現を使っている例を探す

#### パターン2: サイト指定検索

```
"[表現]" site:[サイトドメイン]
```

例:
- "有点儿意思" site:weibo.com
- "挺好的" site:bilibili.com
- "搞定了" site:zhihu.com

**用途**: 特定のプラットフォームでの使用例を探す

#### パターン3: 文脈付き検索

```
"[表現]" + [文脈キーワード]
```

例:
- "有点儿意思" 电影
- "挺好的" 味道
- "搞定了" 工作

**用途**: 特定の文脈での使用例を探す

#### パターン4: 使用場面の検索

```
"[表現]" + 场合/情况/场景/时候
```

例:
- "搞定了" 什么时候用
- "有点儿意思" 使用场景

**用途**: いつ、どんな場面で使うかを調べる

#### パターン5: バリエーション検索

```
[表現A] OR [表現B] OR [表現C]
```

例:
- "挺好的" OR "很好" OR "不错"

**用途**: 似た表現を比較

### 高度な検索テクニック

#### ワイルドカード検索（限定的）

一部の検索エンジンで利用可能：

```
"搞 * 了"
```

**注意**: すべての検索エンジンで対応しているわけではない

#### 除外検索

```
"[表現]" -[除外ワード]
```

例:
- "好吃" -广告 （広告を除外）

#### 期間指定

検索エンジンのツールで期間を指定：
- 過去1年
- 2020年以降
- カスタム期間

**用途**: 最新の使用例、現代的な表現かを確認

## 検索戦略: ステップバイステップ

### ステップ1: 基本検索

まず完全一致検索で全体像を把握：

```
"[表現]"
```

**確認ポイント**:
- 検索結果の数（使用頻度の目安）
- トップに出るサイトの種類
- 見出しで分かる使用場面

### ステップ2: SNSでの使用例を探す

実際の使用例を3-5個探す：

```
"[表現]" site:weibo.com
"[表現]" site:bilibili.com
```

**収集する情報**:
- 実際の文章（前後の文脈を含む）
- 投稿日時
- どんな場面で使われているか
- 感情・ニュアンス

### ステップ3: 使用場面を確認

どんな場面で使うかを明確にする：

```
"[表現]" 场景
"[表現]" 什么时候
```

**確認ポイント**:
- カジュアル/フォーマル
- 口語/書き言葉
- どんな相手に使うか

### ステップ4: バリエーションを探す

似た表現や言い換えを探す：

```
"[表現]" 类似
"[表現]" 同义
[表現A] OR [表現B]
```

**収集する情報**:
- 似た意味の表現
- ニュアンスの違い
- 使い分け

### ステップ5: 評価・選別

集めた用例を評価：

**良い用例の条件**:
- ✅ ネイティブスピーカーの使用
- ✅ 文脈が明確
- ✅ 最近の用例（2020年以降推奨）
- ✅ 信頼できるソース

**避けるべき用例**:
- ❌ 学習者の作文
- ❌ 機械翻訳の可能性
- ❌ 文脈不明
- ❌ 古すぎる（言語は変化する）

## 具体的な検索例

### 例1: 「有点儿意思」の分析

#### 検索クエリ1: 基本検索
```
"有点儿意思"
```

**結果**: 約XXX万件（使用頻度が高い）

#### 検索クエリ2: Weiboでの使用例
```
"有点儿意思" site:weibo.com
```

**見つかった用例**:
1. "今天看了这部电影，有点儿意思，推荐大家去看看！"
2. "这个游戏有点儿意思哈哈"
3. "有点儿意思，没想到是这个结果"

#### 検索クエリ3: bilibiliでの使用例
```
"有点儿意思" site:bilibili.com
```

**見つかった用例**:
1. "这个up主的视频有点儿意思"
2. "有点儿意思，继续看"

#### 検索クエリ4: 使用場面
```
"有点儿意思" 什么时候用
```

**結果**: カジュアルな評価、軽い肯定

#### 分析結果まとめ
- **使用頻度**: 非常に高い
- **場面**: カジュアル、友達同士
- **メディア**: SNS、チャット、動画コメント
- **ニュアンス**: 軽い肯定、「ちょっと面白い」

### 例2: 作文添削での表現検証

#### ユーザーの作文
「这个电影非常好看」

#### 検証: より自然な表現を探す

**検索クエリ1**:
```
"这部电影" "好看" site:weibo.com
```

**見つかった表現**:
- 这部电影挺好看的
- 这部电影真好看
- 这部电影还不错
- 这部电影超好看

**検索クエリ2**: 頻度比較
```
"这部电影非常好看"  →  結果: 少ない
"这部电影挺好看的"  →  結果: 多い
"这部电影真好看"    →  結果: 多い
```

**結論**: 「挺好看的」「真好看」の方が自然

## トラブルシューティング

### 問題1: 検索結果が少なすぎる

**原因**:
- 表現が間違っている
- 非常にまれな表現
- 古い表現

**対策**:
1. 表記を確認（簡体字/繁体字）
2. 類似表現で検索
3. より一般的な部分だけで検索

### 問題2: 検索結果が多すぎる

**原因**:
- 一般的すぎる語
- 複数の意味がある

**対策**:
1. 完全一致検索 "[表現]"
2. 文脈キーワードを追加
3. サイト指定で絞り込み

### 問題3: 信頼できる用例が見つからない

**原因**:
- 学習者の誤用が多い
- 機械翻訳が多い

**対策**:
1. Weibo、bilibiliに絞る
2. Zhihuで議論を探す
3. ニュースサイトで検索

### 問題4: 古い用例しか見つからない

**原因**:
- 表現が古くなっている
- 使用頻度が下がっている

**対策**:
1. 期間を「過去1年」に指定
2. 現代でも使われるか確認
3. 新しい言い方を探す

## 実践的な検索ワークフロー

### ワークフロー: 表現分析モード

1. **基本検索** (2分)
   - "[表現]" で全体像把握
   - 検索結果数を確認

2. **使用例収集** (5-10分)
   - Weibo で 2-3個
   - bilibili で 1-2個
   - Zhihu で 1個（あれば）

3. **文脈分析** (3分)
   - 使用場面を確認
   - フォーマリティを判断

4. **バリエーション探し** (3分)
   - 類似表現を検索
   - 違いを確認

5. **まとめ** (2分)
   - 用例を整理
   - 出典を記録

**合計**: 15-20分

### ワークフロー: 作文添削モード

1. **提案した表現の検証** (3分)
   - 実際に使われているか確認
   - 使用頻度を確認

2. **用例収集** (5分)
   - 2-3個の実用例
   - 同じような文脈

3. **代替案の検索** (3分)
   - より自然な表現はないか
   - バリエーションを探す

4. **まとめ** (2分)
   - 用例を整理
   - 出典を記録

**合計**: 13-15分

## まとめ: 効果的な検索のポイント

### DO（すべきこと）

✅ 完全一致検索 "[表現]" を使う
✅ サイト指定で信頼できるソースに絞る
✅ 文脈付きの用例を探す
✅ 最新の用例を優先する（2020年以降）
✅ 複数の情報源から用例を集める
✅ 出典を必ず記録する

### DON'T（避けるべきこと）

❌ 学習者サイトの用例を鵜呑みにしない
❌ 古すぎる用例（1990年代等）に頼らない
❌ 機械翻訳の可能性がある文を使わない
❌ 文脈不明な用例を使わない
❌ 出典を明示せずに引用しない
❌ 1つの情報源だけに頼らない

### 検索スキル向上のために

1. **練習**: 毎日少しずつ検索スキルを磨く
2. **比較**: 複数の表現を比較検索する習慣
3. **記録**: 良い検索クエリを記録しておく
4. **評価**: 用例の質を評価する目を養う
5. **最新情報**: 新しいプラットフォームもチェック
